# Course Outline

This course provides an introduction to Java programming and fundamental Computer Science concepts. It emphasizes structured problem-solving, logical thinking, and core programming constructs, preparing students for more advanced coursework in Java and Computer Science.

**Topic to cover:**

* Installation Guide

* Computer Literacy Fundamentals

* Program Execution

* Introduction to Programming

  - What is a program?

* Variables

  * Variable concepts and naming conventions

* Data Types

* Useful Java Classes:

  - String Class
    - String concatenation
    - Taking user input

  - Math Class

- Operators and Operands
- Expressions and Statements
- Control Structures
  - Boolean expressions
  - Block statements
  - Conditional statements
  - Iterative statements
  - Basic exception handling
- Functions (Methods)
- Simple Data Structures
  - Arrays
  - Array methods
- Introduction to Object-Oriented Programming
  - Classes and objects
  - Basic OOP concepts
