# Banking System

You are task to simulate a banking software, you will build a robust system by leverage **abstraction**. Your project need to be in a package `bankingSystem`:

1. Create an **interface** `Features`:
   1. Create two methods: `deposit` and `withdrawal`, both will return a Boolean value and accept a double type of amount. (1 mark)
2. Create class called `Account`:
   1. Make the `Account` class implement the `Features` **interface** (1 mark)
   2. Provide an implementation for the unimplemented methods (1 mark)
3. Create a `Tester` class and create a `Account` object (1 mark)
