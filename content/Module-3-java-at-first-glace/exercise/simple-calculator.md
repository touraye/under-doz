# Simple Java Calculator

Assume you have studied **operators, operands, expressions, and statements**.
In this assignment, you will apply your knowledge by performing basic arithmetic operations in Java.

------

### **Objective**

Create a simple Java program that takes two numbers from the user and performs arithmetic operations on them.

------

### **Instructions**

1. Create two variables named `num1` and `num2` of type `double`.
2. Create five additional variables to store results of type `double`:
   - `sum` (addition)
   - `sub` (subtraction)
   - `mul` (multiplication)
   - `div` (division)
   - `mod` (modulus)
3. Prompt the user to enter two numbers using appropriate messages.
4. Use arithmetic operators (`+`, `-`, `*`, `/`, `%`) to perform calculations.
5. Display the result of each operation using separate print statements.

------

### **Expected Output Format**

```
5 + 5 = 10
5 - 5 = 0
5 * 5 = 25
5 / 5 = 1
5 % 5 = 0
```

------

### **Requirements**

- Your code should be **well-structured and readable**.
- Include **meaningful comments** explaining your logic.
- Ensure your program runs **without errors**.
- Handle division by zero safely.

------

**Happy coding!**
