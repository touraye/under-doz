### **Assignment: Investment Contribution Analysis**

Three investors have decided to pool their money together for a period of **one year** with the aim of investing in a lucrative business opportunity.

Each investor must contribute **a minimum of 5,000 and a maximum of 15,000**.

------

### **Objective**

Develop a Java program that collects user input and calculates each investor’s contribution and their percentage share of the total investment.

------

### **Instructions**

1. Define the investment period as **one (1) year**.

2. Use the `Scanner` class to:

   - Prompt the user to enter the **names** of three investors.
   - Prompt the user to enter the **amount each investor contributes**.

3. Ensure each contribution is within the allowed range (**5,000 – 15,000**).

4. Store each investor’s contribution in appropriate variables.

5. Calculate:

   - The **total contribution** of all three investors.
   - The **percentage share** of each investor using the formula:

   ```
   (individual contribution / total contribution) * 100
   ```

6. Display the results for each investor, including:

   - Name
   - Contribution amount
   - Percentage share

------

### **Output Requirements**

- Format monetary values to **two decimal places** (e.g., GMD5,000.00).
- Display percentages with **two decimal places followed by %** (e.g., 33.33%).
- Ensure the output is clear, readable, and well-structured.
