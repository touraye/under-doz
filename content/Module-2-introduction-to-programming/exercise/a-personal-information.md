# Personal Information

## **Exercise: From Personal Information to a Java Program**

### **Objective**

By the end of this exercise, students should be able to:

- Identify different types of personal information.
- Map real-life information to appropriate **data types**.
- Express program logic using **pseudocode**.
- Implement a simple **Java program** using correct data types and variables.

------

## **Part 1: Understanding Yourself as Data (Creative Thinking)**

### **Instructions**

Think of yourself as a *program*. List **at least 8 pieces of personal information** that describe you.

You must include:

- **Textual information** (e.g., name, country)
- **Numerical information** (whole numbers and decimals)
- **Boolean information** (yes/no, true/false)
- **A small collection of related information**

### **Example (Do NOT copy directly)**

- First name
- Age
- Height
- Student status (yes/no)
- Monthly allowance
- Favorite programming languages

------

## **Part 2: Choosing the Correct Data Types**

For each piece of information you listed:

1. Decide the **most appropriate data type**
2. Explain *why* that data type fits the information

------

## **Part 3: Writing Pseudocode**

Now convert your personal information into **pseudocode**.

### **Guidelines**

- Use clear, readable English
- Focus on **declaring variables** and **assigning values**
- Do not use Java syntax yet

------

## **Part 4: Implementing the Java Program**

Using your pseudocode, write a **Java program** that:

- Declares variables using the **correct data types**
- Assigns your personal values
- Prints the information to the console

### **Rules**

- Use meaningful variable names
- Use at least:
  - `String`
  - `int`
  - `double`
  - `boolean`
  - One **collection** (`array` or `ArrayList`)
- The program must compile and run successfully
