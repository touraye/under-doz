Here are four hands-on exercises to practice Java's `Math` class concepts:

---

### **Exercise 1: Temperature Tracker**  
**Goal**: Use `Math.min`, `Math.max`, and `Math.abs` to analyze daily temperatures.  
```java
// Given temperatures: [ -3, 14, 8, -5, 12 ]
// 1. Find the coldest and hottest temperatures.
// 2. Calculate the difference between hottest and coldest temperatures.
// Sample Output: Coldest: -5 | Hottest: 14 | Difference: 19
```

---

### **Exercise 2: Geometry Calculator**  
**Goal**: Use `Math.pow` and `Math.sqrt` to solve geometry problems.  
```java
// 1. Calculate the hypotenuse of a right triangle with sides 5 and 12.
// 2. Compute the volume of a sphere with radius 7 (formula: (4/3)πr³).
// Sample Output: Hypotenuse: 13.0 | Volume: 1436.755...
```

---

### **Exercise 3: Financial Rounding**  
**Goal**: Use `Math.round`, `Math.ceil`, and `Math.floor` for financial calculations.  
```java
// A product costs $19.99. Calculate for 10 items:
// 1. Total cost rounded to nearest integer.
// 2. Total cost rounded *up* (for bulk packaging).
// 3. Total cost rounded *down* (for discounts).
// Sample Output: Rounded: 200 | Ceil: 199.9 | Floor: 199.0
```

---

### **Exercise 4: Dice Game Simulator**  
**Goal**: Use `Math.random` and `Math.sin` for a game scenario.  
```java
// 1. Simulate rolling two 6-sided dice (values 1-6).
// 2. Calculate the launch angle of a projectile (in degrees) 
//    using a random angle between 0-90 degrees.
//    Formula: velocity² * sin(2θ) / gravity (use 9.81 m/s² for gravity).
// Sample Output: Dice: 3 & 5 | Launch Distance: 42.7 meters
```

---

**Key Skills Tested**:  
1. Basic comparisons and absolute values  
2. Powers/roots for geometry  
3. Rounding for financial precision  
4. Random numbers and trigonometry in simulations  

Each exercise mimics real-world scenarios while reinforcing core `Math` class methods.
